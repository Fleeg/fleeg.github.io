About

Fleeg is a project that was written from scratch by Lucas Boscaini with propose to become a social network to index pages. The codename 'fleeg' was chosen because of the mostly common definition of it: "The act of leaving a difficult or boring situation". So, the main goal of this project is to become a large used platform to find contents easily for user in a social network providing a new way to search things on the internet.

Fleeg has just been shared as open source project where everyone can contribute to build that platform as you can and prefer. Right now, it is under construction and some definition are being discussed by maintainers and collaborators.
